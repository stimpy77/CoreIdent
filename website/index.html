Current Status

CoreIdent 1.0 is in active development with a working OAuth/OIDC foundation on .NET 10.

  • Implemented: Token issuance, discovery, JWKS, authorization code + PKCE (with consent), revocation, introspection
  • Implemented: In-memory stores + EF Core stores, refresh token rotation + theft detection
  • Implemented: Test infrastructure + integration coverage for core endpoints
  • Planned next: Passwordless flows (email magic links, passkeys), providers, client libraries
